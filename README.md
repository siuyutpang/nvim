# nvim

firstly, you should select a package manager for your neovim.(e.g. [vim-plug](https://github.com/junegunn/vim-plug))

```
curl -fLo "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim/autoload/plug.vim --create-dirs \
       https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

secondly, you can open neovim, then press **Shift + ;**  keywords, input PlugInstall command to install plugin.
